Required parameters

For each operation, you have to give the following required parameters are * Type => it must be a manage_entity * Action => create/delete. it must be in the action array in your manage_entity * Remainning parameters => contains all the data you need to create/delete an entity

Available routes

  • Batch : GET /batches/{id}, POST /batches, DELETE /batches/{id}
  • Operation : GET /operations/{id}

For more detailled options for these routes, you can use nelmioApiDocBundle. our routes are commented, and you will find the documentation.
